# Лабораторна робота №11 — Рядки

Тема: робота з рядками, `ListBox`, `Label`, `StringBuilder`, методами `IndexOf`, `Substring`, `Replace`, `Split`, `Regex`.

## Що зроблено

- Windows Forms застосунок C#.
- Рядки вводяться через `ListBox`.
- Результат виводиться у `Label`.
- Є можливість додати власний рядок у список.
- Реалізовано аналіз рядка:
  - кількість слів;
  - кількість цифр;
  - кількість знаків пунктуації;
  - виведення цифр;
  - розділення на цифри, пунктуацію та інші символи;
  - пошук малих українських букв;
  - перевірка паліндрому;
  - заміна місцями першого та останнього слова.

## Як запустити

1. Відкрити файл `Autosalon_Lab11_Strings.csproj` у Visual Studio.
2. Натиснути `Start` або клавішу `F5`.
3. Обрати рядок у `ListBox`.
4. Натиснути потрібну кнопку обробки.
